# RedNote Signals

Use this checklist when the user records Xiaohongshu/RedNote searches, posts, comments, saves, screenshots, or browsing memories.

## Extract

- search keywords
- titles that made the user stop
- cover text patterns
- first-line hooks
- comment-section questions
- saves/likes/follows and why
- repeated desires:省钱, 变美, 起号, 副业, 模板, 教程, 避坑, 情绪共鸣
- repeated fears:怕贵, 怕麻烦, 怕失败, 不知道从哪开始
- formats:清单, 对比, 教程, 日记, 复盘, 反差, 前后变化

## Turn Observations Into Ideas

Prefer ideas that combine:

- a repeated keyword
- a real user question
- a specific format
- the user's own curiosity or experience

Example:

```text
观察：反复看到“普通人起号”“封面模板”“没人点”。
机会：做一篇“普通人小红书没人点，先别急着怪内容”的拆解。
```

## Keep Judgment Light

Do not call every saved post a "爆款机会". Mark uncertainty:

- "值得继续观察"
- "可能能做成选题"
- "先用 3 条类似笔记验证"
