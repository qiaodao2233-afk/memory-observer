# Memory Observer

Memory Observer is a concise personal memory and RedNote/Xiaohongshu observation skill for Codex.

It helps a user:

- record what they watched, searched, noticed, and thought about
- make short daily and weekly summaries
- extract RedNote/Xiaohongshu content signals
- give brief encouragement or comfort
- support optional weather, blessing, and voice-assistant reminders

The skill is designed to be quiet: report clearly, then stop.

## Install

Upload this folder or a zip of this folder as a Codex Skill.

Required files:

```text
SKILL.md
agents/
references/
```

## Privacy

Do not upload personal memory folders.

Keep these local or private:

```text
.private/
memory-observer-data/
```
